Create New Task

  • With a task you can track your productivity and progress toward a specific goal.
  • Write a description of the work and preparation you'll need to perform in order to complete the task.
  • Designate the task status, consider whether this task will need to wait on outside information or if its already been started.
  • Assign a number of points the task will need. Read below to see how points work.
  • After you submit your new task, you'll be able to find it in your team's backlog.
